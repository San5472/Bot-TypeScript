# Bot-TypeScript
Crear un Bot usando TypeScript


# Resultados del BOT

Al colocar el comando **/help** en la consola arrajo lo siguiente

    Yo >/help

    bot > Comandos disponibles:
    /help -> muestra esta ayuda
    /ping -> muestra que el bot responde
    /echo -> repite lo que escribas


Al colocar el comando de **/ping** en la consola se muestra lo siguiente

    Yo >/ping
    bot > Latencia estimada: 0ms

Por ultimo al momento de colocar el ultimo comando **/echo** muestra el mensaje que se le coloca

    Yo >/echo bot de prueba, para conocer un poco más de la syntaxis de TypeScript y mejorar un poco la logica 
    bot > bot de prueba, para conocer un poco más de la syntaxis de TypeScript y mejorar un poco la logica
